Compatibility

Xcode

Swift

5.1 - 5.1.3

iOS

  • Fully supported: iOS 11.0 - 13.3 (see note below)
  • Unsupported, but will install/run: iOS 9.3.5 - 10.3.4

Note: dark mode on iOS 13.x is currently not supported.
