AchievementFeedbackModalManager

@objc(SBLAchievementFeedbackModalManager)
public class AchievementFeedbackModalManager : NSObject

Helper Class to control manage first time experience feedback modals. This class will be used to track whether a welcome experence modal should be shown, and the content in the modal.
